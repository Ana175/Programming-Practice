a. Below is the list of the programs that enables you to enhance your knowledge about loops(for/while) and conditional statements(if-else/switch).

     1. Write a program that prints a fibonaci series that is a sequence of numbers like0 1 1 2 3 5 8.You can vary the number of elements to be printed meaning you can print 10 numbers or 15 or 20 or any desired number.
     2. Write a program print "ping" if a number is divisible by 3,"pong" if a number is divisible by 5, and "ping pong" if number is divisible by both, else print the number.
     3. Write a program that swaps 2 given numbers. You need to have 2 separate functions in the program.
         --> One Function should swap the numbers without any third new variable.
         --> Second function should swap the numbers using a third variable.
     4. Write a program to calculate factorial of a given number. You need to compute the factorial with one logic that uses recursion and another logic without recursion.
     5. Write a program to check if a given number is prime or not.
     6. Write a program to check an armstrong number i.e. whether the if we power up the each individual number to the total number of digits in the number and add them it should be equal to the number itself. For example 153 has 3 digits in it and if we do 1^3+5^3+3^3=153.
     FYI: ^ means is to the power.
     7. Write a program that accepts input from the command line and then prints them.
     8. Write a program that prints a pattern like below.
         *
         **
         ***
         ****
         *****
         ******

b. Below is a list of programs on java arrays. Candidates are instructed not to use any inbuilt functions.

     1. Write a program to find the largest number in a given array.
     2. Write a program to find the missing number in a series of sorted numbers stored in an array.
     3. Write a program to find the common number in any given two arrays.
     4. Write a program to perform a linear search on any given array.Linear search is the basic search where you look for the element to be searched in a sequential way.
     5. Write a program to sort an array.
     6. Write a program to locate and swap only 2 elements which are not sorted in a given sorted array. For example given an array {1,2,5,6,4} locate 5 and 4 and then swap them as they are not as per sorted order.
     7. Write a program to merge 2 arrays.

c. Below is a list of programs on String computations.

     1. Write a program to reverse a string.Donot use StringBuffer/StringBuilder inbuilt reverse function.
     2. Write a program to find a substring in a given string and then replace it with another string
     3. Write a program which accepts a string like "This is nice" and converts it to a string like "This1 is2 nice3".
     4. Write a program to reverse each individual word in a sentence.
     5. Write a program to check for a palindrome string.
     6. Write a program to check if a given string of parenthesis has balanced parenthesis or not.For example a string as "(())" is a valid string where as strings like ")(" or "(()))" are not valid strings.
     7. Write a program to implement hashCode() and equals() methods.
